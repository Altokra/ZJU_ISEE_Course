## 体验

成绩构成：平时50%+期末大作业50%

物联网的知识相对比较好理解，实验也没有太复杂，只要跟着ppt做就能完成；想理解地深入一些可以听老师讲。

不点名，但作为一门实验课，耗时比较长。先在西教上两课时的理论课，再去东教做实验，还有每周一次的实验报告。不过相对于大二上的每周大物实验+电设实验的两份报告还是轻松不少((

期末大作业两人一组，自选题目做一个物联网系统，基本上就是根据课程内容延申，想做什么自己加，实现难度应该都不大。

有问题直接问老师

>[@xsdzz](https://www.cc98.org/topic/6217373):ldw老师和另一位老师qcl都是非常负责有耐心的老师，遇到不会的问题老师都会很耐心的解答，而且两位老师对学生还挺宽容的，作业如果忘记交或者交不上老师会给开通道补交，特别是qcl老师，是我在信电见过最温柔有耐心的老师。老师对学生的作业也很负责，做的所有作业老师都会看并且给出评价和建议。

于我个人而言，最大的收获是巩固latex和git的知识，并逐渐形成自己的工作流。至于嵌入式和云端联动的一些知识，之前都接触过一些，就没有太多初学的经验分享了。

分享资料只有实验报告，其余内容上课基本都会提供，由于版权等问题就不放在这了。
